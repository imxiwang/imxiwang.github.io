续费夏天——地中海！

发表于 2019-11-28

提前给大家道个歉，本来是四号去的特拉维夫，结果现在才发出来。各种事情实在是太多了啊哈哈哈~
话不多说进入正题！特拉维夫实在是太美了！

  难得实验课轮班，今天放了一天假，于是我们临时决定去一趟特拉维夫。巴伊兰大学距离特拉维夫只有五六公里的样子，也就是说去海边的距离和从武大到长江边差不多。如此良机，怎能错过？
  于是，早上十点，我们到达了这个海滨城市。


  路边的咖啡馆随处可见。其实，像咖啡馆这种文化现象可以在这里流行，气候因素占得挺大。因为这里虽然热，但是如果能找到一个遮阳的地方，那效果和开了一个小空调差不太多。如果是在武汉的夏天，难以想象。
  转了几个街道之后，我可以断定特拉维夫是世界上生活最安逸的几个地方之一了。

  下面貌似是一个很标志性的位置，旁边有一群大妈在穿着旗袍（？）拍照。话说以色列对于喷泉有一种超乎寻常的狂热，学校里面也有好多小型喷泉景观。
  本来我应该出镜的，但是由于本人不太好意思，拍照留念这种事就由书包代劳啦~

  中午找到了一家中餐厅，吃到了久违的中国菜。青菜的味道实在是令我感动，它的价格更甚——一百RMB一盘。在国内，如果这钱光吃青菜，已经可以吃到幸福地落泪了。
  不过老板人挺好，貌似是从广东那边过来谋生的。吃饭的时候还碰见了一群国内因公出差的叔叔阿姨，对我们说：多吃一点，爸爸妈妈才会高兴！

  水饱饭足之后，我们朝海边进发。一路上街边的房子都很精致，人们也十分的悠闲，我们有说有笑地向前走。如果能在这里养老该多好！
  离海还有百米的时候，我越发感到了一种难以言喻的愉悦，这来源于内陆人民对于大海长久的渴望。空气愈发清新，精神也愈发亢奋，仿佛将要得到真正的自由。
  终于！看到了！那过去只存在于白日梦和黑夜梦里的地中海！

  天哪！
  一个湖北人，接触到了真正的海。
  海水很清凉。沙子被海水冲刷地很结实，而沙子上的脚印一会儿就被海浪抚平了。海浪一股劲冲过来，有时候会把站在浅水里的人猝不及防地摇晃一下。潜水坑里还有机灵的鱼。

  上岸之后继续逛了几个街道，走到了雅法。我手里只有最后几张照片啦，带大家看一看。






  好啦，接下来就是压轴大戏——约拿的🐟！我亲手摸到了它的尾巴。

心满意足，返回学校，继续追赶ddl！
